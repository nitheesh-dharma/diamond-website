Features and Facilities

The top floor of the diamond has several features and facilities. To start with, it has a library with a plethora of books to read as well as places to study and read within the library itself. Furthermore, there are several quiet places to study, some of them even contain computers to do work. There are also media rooms which can be used to compose and make music. There are also group meeting rooms which students can use to work on group assignments. For more information on the top floor click here and look at the description for level 4.
